# So far just supports for Hyperledger Fabric

## Usage
You may need to go to Tape to see details about usage.
For basic asset, we have sample config as
```shell
cd ./Traffic/fabric
./tape.sh
```
For NFT, we have sample config as
```shell
cd ./Traffic/fabric
./tape.sh NFT
```
